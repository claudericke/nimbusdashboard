# Product Roadmap

1. [ ] **Dashboard Overview Bento Grid** — Implement the main dashboard view with widgets for Server Status, Active Package, Weather, Local Time, disk usage, and SSL status. `[M]`
2. [ ] **Email Management Interface** — Create a simplified view for users to add, remove, and manage email accounts for their domains. `[M]`
3. [ ] **Domain Management & cPanel Auth** — Implement core domain viewing and user password update functions via cPanel API. `[M]`
4. [ ] **User Onboarding System** — Develop the "Add User" function that triggers the automatic onboarding email using `onboardingEmail.md`. `[S]`
5. [ ] **Trello Ticket Integration** — Connect to Trello API to display open support tickets and allow users to close them from the dashboard. `[S]`
6. [ ] **Zoho Billing Integration** — Implement billing management by fetching and displaying invoices from the Zoho API. `[M]`
7. [ ] **Admin Domain & User Management** — Build the administrative interface for managing domains and users across all accounts. `[M]`
8. [ ] **UI Polish & Animations** — Refine the interface with Bootstrap and integrate animation libraries for a premium feel. `[S]`

> Notes
> - Foundational MVC structure is already in place.
> - Dashboard widgets are the primary entry point for users.
> - API integrations (cPanel, Trello, Zoho) are critical for end-to-end functionality.
